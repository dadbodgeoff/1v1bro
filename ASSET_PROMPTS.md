## Asset Specifications Summary

| Asset Type | Size | Format | Quantity |
|------------|------|--------|----------|
| Player sprites | 32x48 per frame | PNG (transparent) | 2 characters × 32 frames |
| Power-up icons | 64x64 | PNG (transparent) | 4 icons |
| Floor tiles | 256x256 | PNG (seamless) | 1-3 variations |
| UI icons | 64x64 | PNG/SVG | 10-15 icons |
| App icon | 512x512 | PNG | 1 |
| Logo | Vector | SVG | 1 |
| Effect sprites | 128x128 per frame | PNG (transparent) | 4-6 effects × 8 frames |
| Backgrounds | 1920x1080 | PNG/JPG | 3-5 |

## Recommended Tools

### Image Generation
- **Midjourney** - Best for stylized game art
- **DALL-E 3** - Good for icons and UI elements
- **Leonardo AI** - Great for game assets specifically
- **Stable Diffusion** - Free, customizable with LoRAs

### Sprite Sheet Tools
- **Aseprite** - Pixel art animation
- **TexturePacker** - Combine sprites into sheets
- **Piskel** - Free browser-based pixel art

### Audio
- **Soundraw** - AI music generation
- **Eleven Labs** - Voice/sound effects
- **Freesound.org** - Free sound effects library

## Style Guide Reference

Keep all assets consistent with:
- **Primary colors:** Lime #00ff88, Pink #ff006e, Cyan #00d4ff
- **Background:** Dark navy #0a0e27
- **Accent:** Purple #b300ff, Gold #ffb700
- **Style:** Cyberpunk, neon glow, clean edges
- **Mood:** Competitive, energetic, fun

---

*Generate assets, drop them in `frontend/public/assets/` and update the renderers to use images instead of canvas shapes.*
